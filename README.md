# 바로빌 테마 (Borobill Theme)

WordPress 테마 프로젝트입니다.

## 프로젝트 구조

```
borobill_theme/
├── header.php          # 헤더 템플릿
├── footer.php          # 푸터 템플릿
├── index.php           # 메인 템플릿
├── single.php          # 단일 포스트 템플릿
├── archive.php         # 아카이브 템플릿
├── functions.php       # 테마 함수
├── style.css           # 스타일시트
├── js/
│   └── main.js         # 메인 JavaScript
└── images/             # 이미지 파일들
```

## 설치 방법

1. WordPress 테마 디렉토리에 업로드
2. WordPress 관리자에서 테마 활성화
