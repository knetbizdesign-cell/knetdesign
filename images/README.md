# 이미지 폴더

이 폴더에 테마에서 사용할 이미지를 넣어주세요.

## 폴더 구조

```
images/
├── hero-bg.jpg          # 히어로 섹션 배경 이미지
├── logo.png             # 바로빌 로고
└── placeholder.jpg      # 플레이스홀더 이미지
```

## 사용 방법

### 1. 정적 이미지 (배경, 로고 등)
- 이 폴더에 이미지를 넣고
- 코드에서 `<?php echo get_template_directory_uri(); ?>/images/파일명.jpg` 로 사용

### 2. 포스트 썸네일
- WordPress 관리자 > 미디어 라이브러리에서 업로드
- 각 포스트 편집 시 "특성 이미지" 설정

### 3. 배너 이미지
- 이 폴더에 넣거나 미디어 라이브러리 사용 가능
