# Figma 디자인을 WordPress 테마로 정확히 구현하는 가이드

## 단계별 구현 방법

### 1단계: Figma에서 정확한 값 추출

#### 필요한 정보:
- **좌표 (x, y)**: 각 요소의 정확한 위치
- **크기 (width, height)**: 각 요소의 정확한 크기
- **색상 코드**: HEX 값 (#5d88b9 등)
- **폰트 스타일**: 폰트 패밀리, 크기, 굵기, line-height
- **간격 (gap, padding, margin)**: 요소 간의 정확한 간격

### 2단계: 레이아웃 구조 분석

#### 전체 구조 (Figma 기준):
```
전체 프레임: 1920px × 3382px
├── 헤더: x:418, y:12.602, width:1084, height:35
├── 히어로: x:421, y:94, width:1078, height:379
├── 추천 게시물: x:418, y:535, width:1084, height:441
├── 카테고리 필터: x:425.556, y:1112, width:515, height:42
├── 게시물 목록: x:426, y:1224.276, width:1038, height:1216
├── 배너: x:421, y:2659, width:1078, height:191
└── 푸터: x:320, y:2940, width:1280, height:466
```

### 3단계: CSS로 정확한 위치 설정

#### 방법:
1. **절대 위치 사용 (필요시)**: `position: absolute` + `left`, `top`
2. **Flexbox/Grid 사용**: 대부분의 레이아웃
3. **정확한 크기 지정**: Figma 값 그대로 사용

#### 예시:
```css
/* Figma: x:421, y:94, width:1078, height:379 */
.hero-section {
    margin-left: 421px;
    margin-top: 94px;
    width: 1078px;
    height: 379px;
}
```

### 4단계: 색상과 폰트 정확히 매칭

#### 색상:
- 히어로 배경: `#5d88b9`
- 배너 배경: `#5782ef`
- 텍스트 기본: `#0b0b0e`
- 텍스트 보조: `#505050`
- 카테고리 배지: `#e4e8ed` / `#2d4a62`

#### 폰트:
- 폰트 패밀리: `Pretendard`
- 폰트 크기: Figma에서 정확한 값 사용 (예: 45.02751922607422px)
- 폰트 굵기: 600 (SemiBold), 700 (Bold)
- Line-height: Figma 값 그대로 사용

### 5단계: 이미지 적용

#### 이미지 위치:
- `wp-content/themes/borobill_theme/images/` 폴더에 저장
- 히어로 배경: `1.png`
- 게시물 썸네일: `1.png` ~ `10.png` (순환 사용)
- 로고: `logo.png`, `logo2.png`
- 소셜 아이콘: `Naver.png`, `Youtube.png`

### 6단계: 반응형 대응

#### 브레이크포인트:
- 1920px 이상: Figma 디자인 그대로
- 1440px 이하: 간격 조정, 그리드 2열
- 768px 이하: 모바일 레이아웃 (세로 배치)

## 현재 구현 상태

✅ 완료된 항목:
- 헤더 구조 및 스타일
- 히어로 섹션 기본 구조
- 추천 게시물 그리드
- 게시물 리스트 레이아웃
- 카테고리 필터
